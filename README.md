# 🧠 Hack The Box Labs - Miguel Tejada 🤓

Repositorio de prácticas y resolución de laboratorios en Hack The Box.  
Enfocado en desarrollar habilidades reales de **pentesting, enumeración, explotación y post-explotación**.

---

## 🎯 Objetivo

Construir un portafolio técnico basado en:

- Metodologías reales de pentesting (Recon → Exploit → PrivEsc)
- Documentación profesional (writeups claros y estructurados)
- Análisis de vulnerabilidades y defensas (HTB Inverso)

## 🧠 Metodología

Cada laboratorio sigue esta estructura:

1. **Enumeración**
2. **Explotación**
3. **Post-Explotación**
